# Sitzey

Marketing site for Sitzey, a website design & development studio for small
businesses. Dark, glassmorphic design system with a purple/fuchsia glow
palette, a Spline 3D hero, and a shared scroll-reveal/magnetic-hover
animation system.

## Stack

- [Next.js](https://nextjs.org) 16 (App Router, Turbopack)
- React 19 · TypeScript (strict)
- Tailwind CSS v4 + [shadcn/ui](https://ui.shadcn.com) conventions (`components.json`, `@/components/ui`)
- [lucide-react](https://lucide.dev) for icons
- [Spline](https://spline.design) for the hero's 3D background
- `motion` (only used by the glowing-border hover effect)

## Routes

- `/` — home (hero, why-a-website, services/pricing, process)
- `/contact` — contact form, FAQ, alternate contact methods
- `/robots.txt`, `/sitemap.xml`, `/opengraph-image` — generated by `src/app/robots.ts`, `sitemap.ts`, `opengraph-image.tsx`

## Development

```bash
npm run dev     # start the dev server at http://localhost:3000
npm run build   # production build
npm run start   # run the production build
npm run lint    # eslint
```

## Project structure

- `src/components/blocks/` — page sections and decorative/animated components specific to this site (nav, hero, footer, forms, background effects)
- `src/components/ui/` — generic shadcn primitives (`Button`, `GlowingEffect`)
- `src/lib/utils.ts` — shared helpers (`cn`, `focusRing`)
